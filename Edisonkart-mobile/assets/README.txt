Add these files for the app to build:
- icon.png (1024x1024)
- splash-icon.png (recommended 1284x2778)
- adaptive-icon.png (1024x1024, Android)
- favicon.png (48x48, web)

Or run: npx create-expo-app@latest temp --template blank && copy temp\assets\* . && rmdir /s /q temp
(On Mac/Linux use cp and rm -rf instead of copy and rmdir.)
